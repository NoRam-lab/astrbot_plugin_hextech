# AstrBot Hextech Plugin (海克斯乱斗信息差)

这是一个用于查询《英雄联盟》极地大乱斗（ARAM）英雄海克斯联动数据的 AstrBot 插件。数据来源于 [Apexlol](https://apexlol.info/)。

## 主要功能

### 1) 智能英雄识别

- 支持标准英雄名、常见别名、外号、数字梗输入（如 `4396`、`快乐风男`）。
- 可选使用 LLM 将输入标准化为英雄官方名称。

### 2) 海斗图片报告（新）

`/海斗 <英雄名或别名>` 现在会生成并发送图片报告（成功时只发图片）：

- 英雄名称与称号
- 英雄介绍
- 技能组：被动 + Q/W/E/R（含技能图标）
- 海克斯联动分析 Top 10（含联动图标、评级、阶级、说明）

### 3) 海克斯文本查询

`/海克斯 <海克斯名>` 会返回文本检索结果（适合快速查单个符文说明）。

## 安装

1. 将插件放置在 AstrBot 插件目录。
2. 安装依赖：

```bash
pip install -r requirements.txt
```

或手动安装：

```bash
pip install aiohttp beautifulsoup4 Pillow
```

1. 重启 AstrBot。

## 配置

插件支持以下配置项：

- `enable_llm_search`（默认 `true`）
  - 是否启用 LLM 英雄名标准化。
- `llm_provider_id`（可选）
  - 指定 LLM Provider；不填时会尝试使用第一个可用 provider。

如果没有可用 LLM Provider，插件仍可通过本地英雄库进行匹配（但别名识别能力会下降）。

## 使用方法

### 指令

```text
/海斗 <英雄名或别名>
/海克斯 <海克斯名>
```

### 示例

```text
/海斗 诺手
/海斗 4396
/海克斯 利刃华尔兹
```

### 触发说明

插件同时支持：

- `@filter.command` 指令触发
- 被动监听兜底（当部分环境 command 触发不稳定时）

因此在不同适配器环境中，`/海斗` 和 `/海克斯` 都有更高概率稳定触发。

## 图片链路说明

`/海斗` 的图片生成与发送流程：

1. 请求英雄页面并解析结构化内容。
2. 使用 HTML 渲染生成图片。
3. 使用 Pillow 对渲染结果做有效性校验（防止返回非图片内容）。
4. 图片落盘后通过 `Image.fromFileSystem(...)` 发送。

## 常见问题

### 1) 返回“获取英雄页面失败”

通常是网络不可达或目标站点临时不可用。请检查宿主机网络是否可访问 `https://apexlol.info/`。

### 2) 返回“图片生成失败”

通常是渲染服务不可用（`html_render`/T2I 链路异常）。请检查 AstrBot 渲染配置与服务状态。

### 3) 返回“图片发送失败”

通常是平台发图链路问题（适配器/NapCat/文件可见性）。请检查日志中的图片路径与文件大小。

### 4) 技能或联动数量不足

页面结构变动会影响解析。插件已做多选择器兜底；若持续出现，请反馈页面示例链接以便更新解析逻辑。

## 注意事项

- 本插件数据来自第三方网站，仅供参考。
- 英雄与联动数据依赖网页结构，目标网站结构变更时可能需要同步更新插件。
